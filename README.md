Github User Search
==================

> Lookup for a github user through its API

## Download
Download in current directory
```sh
$ npm install
```

## Usage
```javascript
new SearchGithubUser({opts});
```

## Options
- Input
```javascript
input: domNode
 ```
 - Results Container
 ```javascript
resultsContainer: domNode
 ```
 - Search title
 ```javascript
searchTitle: string
```
